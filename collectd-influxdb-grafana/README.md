## Monitoring system:

* Used tools:

* Collectd
* InfluxDB
* Grafana

###1. Build with Docker Compose:
```docker-compose build```

###2. Run with Doker Compose:
```docker-compose up -d```

###3. Kill running cointainers:
```docker-compose kill```

###4. Remove stopped cointainers:
```docker-compose rm -f```
